    Projects

    Software Development Internship

      In this internship, I developed a software to help mining industry to get particle size distribution curve automatically. After I presented this software to investors of our institute, we got funding from investors to continue this project!

       

      In mining industry, it is necessary to get particle size distribution curve to measure stability of waste rock. They use this information combined with chemistry and geology theory to calculate if stone pile can be reused. Usually, they measure it manully which has low precision and vey time consuming.

       

      With my software, they can use phone to take photos and use this software to get particle size distribution curve. This software helps them reduce more than 80% labor cost!
